穴埋め解除ビューア

先生が答えの上に乗せた「白い長方形」を消したり、「白文字（見えない文字）」に色を付けて、 隠された答えをブラウザ内で表示するツールです。

穴埋め解除

白い長方形の上にカーソルを重ね、枠が出た所をクリックすると、その被せ物だけを消せます。もう一度クリックで戻せます。

感度

被せ物や白文字が消えない / 色が付かないときは、値を下げて薄いグレーも対象にしてください。

表示

編集

検出した被せ物 0 個 / 表示中 0 個

メモ

処理はブラウザ内だけで行われ、PDFはどこにも送信されません。被せ物の検出は白い塗りつぶしと白い文字が対象です。 画像で隠されている場合や、複雑なPDFでは検出できないことがあります。

PDFを読み込んでください

ローカルPDFを選択するか、このリポジトリに含まれる main.pdf をサンプルとして開けます。
