穴埋め解除ビューア

GitHub Pages

白い被せ物を消す・白文字に色を付けて、隠された答えをブラウザ内で表示します。

穴埋め解除

被せ物と白文字の表示設定

白い長方形にカーソルを重ねてクリックすると、その被せ物だけを消せます。もう一度クリックで戻せます。

感度

白とみなす明るさのしきい値

検出されない場合は値を下げて、薄いグレーも対象にしてください。

編集

文字・画像の追加とPDF結合

文字 0 画像 0

編集モードを選ぶと、PDF上で文字をクリックして変更するか、画像を配置できます。穴埋め解除のクリック操作は一時的に無効になります。

別のPDFの全ページを現在の文書の末尾に結合します。

表示

プレビューの拡大率

操作

穴埋め解除のリセット

検出 0 表示 0

穴埋め解除の設定（被せ物の表示・白文字の色付け）のみリセットします。文字・画像の編集はヘッダーの元に戻すで取り消せます。

メモ

処理はブラウザ内だけで行われ、PDFは送信されません。画像で隠された場合や複雑なPDFでは検出できないことがあります。

PDFを読み込んでください

ローカルPDFを選択するか、リポジトリの main.pdf をサンプルとして開けます。
